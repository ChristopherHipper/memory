🧠 Memory Game

An interactive memory card game with multiple themes, built using **TypeScript**, **SCSS**, and **Vite**.

🎮 Features

* 🃏 Classic memory gameplay
* 🎨 Multiple themes (e.g. coding, game and food.)
* ⚡ Fast performance powered by Vite
* 🎯 Game logic implemented in TypeScript

🛠️ Tech Stack

* **TypeScript** – for clean and scalable logic
* **SCSS** – for structured and maintainable styles
* **Vite** – for fast development and optimized builds

🚀 Getting Started

```bash
# Clone the repository
git clone https://github.com/ChristopherHipper/memory.git

# Install dependencies
npm install

# Start development server
npm run dev
```

🏗️ Build

```bash
npm run build
```
